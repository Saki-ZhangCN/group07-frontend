# 在线教育智慧教学平台 - 设计系统

## 一、设计理念

### 1.1 设计目标
- **现代简约**: 采用简洁的卡片式布局，减少视觉干扰
- **渐变美学**: 使用蓝紫渐变作为主色调，传递科技感与教育属性
- **动效丰富**: 添加适度的微动画提升用户体验
- **响应式设计**: 适配桌面端和移动端

### 1.2 核心价值观
- 清晰的信息层级
- 舒适的视觉体验
- 流畅的交互反馈

## 三、排版规范

### 3.1 字体栈

```css
font-family: 'Inter', 'Noto Sans SC', system-ui, sans-serif;
```

### 3.2 字号层级

| 层级 | 字号 | 用途 | 示例 |
|-----|------|------|------|
| H1 | 3xl (30px) | 页面标题 | `text-3xl font-bold` |
| H2 | 2xl (24px) | 区块标题 | `text-2xl font-semibold` |
| H3 | xl (20px) | 卡片标题 | `text-xl font-medium` |
| Body | base (16px) | 正文内容 | `text-base` |
| Small | sm (14px) | 次要信息 | `text-sm` |
| Caption | xs (12px) | 辅助说明 | `text-xs` |

### 3.3 字重规范

- **Bold (700)**: 标题、重要数据
- **Semibold (600)**: 卡片标题、按钮文字
- **Medium (500)**: 普通文字、标签
- **Regular (400)**: 正文、描述文字

## 九、间距规范

### 9.1 基础间距

| 间距值 | 用途 |
|-------|------|
| 4px (0.25rem) | 内边距、小间距 |
| 8px (0.5rem) | 元素间距 |
| 12px (0.75rem) | 中等间距 |
| 16px (1rem) | 标准间距 |
| 24px (1.5rem) | 大间距 |
| 32px (2rem) | 区块间距 |

### 9.2 间距原则

- 使用 4px 的倍数作为间距单位
- 保持视觉节奏的一致性
- 卡片内边距统一为 20px

## 十四、总结

本设计系统包含了在线教育智慧教学平台的完整视觉规范，涵盖颜色、排版、组件、布局和动画等方面。使用此文档可以快速搭建具有一致风格的界面。

**核心设计特点：**
1. 渐变蓝紫主色调，传递科技教育感
2. 卡片式布局，信息层次清晰
3. 丰富的微动画，提升交互体验
4. 响应式设计，适配多端设备
5. 统一的设计语言，保持视觉一致性
